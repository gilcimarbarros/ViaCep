# ViaCep
 - App simples  de consulta de Cep consumindo a API ViaCep através da biblioteca Retrofit
 - Com as funções de consulta e de limpeza dos campos
 - Exibe mensagem "Informe o Cep"
 - Exibe mensagem "Cep invalido"
